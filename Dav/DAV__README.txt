# DAV_Practical/README.txt
Data Analytics and Visualization Practical Exam Project

File Mapping:
q1_eda_python.py - EDA in Python
q2_eda_R.R - EDA in R
q3_simple_lr_python.py - Simple Linear Regression in Python
q4_simple_lr_R.R - Simple Linear Regression in R
q5_multiple_lr_python.py - Multiple Linear Regression in Python
q6_multiple_lr_R.R - Multiple Linear Regression in R
q7_time_series.py - Time Series Components and Stationarity (Python)
q8_arima.py - ARIMA, ACF/PACF, and Forecasting (Python)
q9_nlp.py - NLP Preprocessing (Tokenization, Stopwords, Stemming, Lemmatization)
q10_bow.py - Bag of Words representation (Python)
q11_tfidf.py - TF-IDF representation (Python)
dataset.csv - Sample data used for Question 1

Execution Instructions:
1. Ensure Python dependencies are installed: `pip install -r requirements.txt`
2. NLTK data (punkt, stopwords, wordnet) will auto-download on first run of q9_nlp.py.
3. R scripts use built-in datasets (mtcars, cars) and require no extra installations. Run in RStudio.
4. All Python code is executable instantly in VS Code, Jupyter, or Google Colab.
